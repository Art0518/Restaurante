?? Variables Globales Definidas en carrito.js

  • reservasSeleccionadas = [] - ? Lista de reservas seleccionadas por el usuario
  • todasReservas = [] - ? Todas las reservas del carrito
  • promocionSeleccionada = null - ? Promoción actualmente seleccionada
  • promocionesDisponibles = [] - ? Lista de promociones disponibles
  • facturaActual = null - ? Factura actualmente mostrada
  • usuario = null - ? Datos del usuario logueado

?? Funciones que Usan Variables Corregidas

  • actualizarSeleccion(): ? Usa correctamente reservasSeleccionadas
  • seleccionarTodasReservas(): ? Maneja reservasSeleccionadas
  • deseleccionarTodasReservas(): ? Resetea reservasSeleccionadas
  • confirmarReservasSeleccionadas(): ? Envía reservasSeleccionadas
  • generarFacturaDesdeCarrito(): ? Utiliza reservasSeleccionadas

  4. Intenta seleccionar una reserva
